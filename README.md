# Nutrition Optimization API

## 概要
目標とするマクロ栄養素（タンパク質、脂質、炭水化物）を満たすための最適な食材の分量を算出するバックエンドAPIです。

## 技術スタック
- **Python 3.12**
- **FastAPI**: 型安全性と高速なレスポンスを実現するため採用。
- **NumPy**: 最適化計算の要となる行列演算エンジンとして採用。
- **SQLite3**: 外部のBaaSに依存せず、スタンドアロンで動作するアーキテクチャとするため採用。

## 計算ロジック
単純な条件分岐（if文）による加算ではなく、線形代数を用いたアプローチを実装しています。
各食材の栄養素を行列 $A$、摂取量をベクトル $x$、目標栄養素をベクトル $b$ と定義し、$Ax = b$ の連立一次方程式に対して、最小二乗法（`numpy.linalg.lstsq`）を適用することで、理論上最も誤差の少ない最適な分量を数学的に導き出しています。

## 起動手順
1. リポジトリをクローン
2. 仮想環境の構築: `python -m venv venv`
3. 依存ライブラリのインストール: `pip install -r requirements.txt`
4. データベースの初期化: `python database.py`
5. サーバーの起動: `uvicorn main:app --reload`
6. `http://127.0.0.1:8000/docs` にアクセスしてSwagger UIからテストを実行
